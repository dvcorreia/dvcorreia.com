Site powered by [Hugo](https://gohugo.io) and hosted by [Cloudflare Pages](https://pages.cloudflare.com/).

The style is a never-ending cycle of change that always ends up in frustration. Currently, it's stolen from [mattwidmann.net](https://mattwidmann.net), which I apologize for. With my lack of design skills, I might resort to contracting a designer to help out.

I use CSS with no preprocessor other than [PostCSS](https://postcss.org/) to inline `@import` at-rules, so I don't get lost in a single massive CSS file.

Haven't dedicated much effort to make the site accessible, but throughout the coding process I tried to do as much as possible.

The website does use Javascript but works just fine without it for all the JS-free folks. You'll be able to read all the content, but certain JS-powered features will either be invisible or unavailable.

It doesn't depend on resources from other domains, with the exception of Cloudflare's injected script for [Email Address Obfuscation](https://developers.cloudflare.com/waf/tools/scrape-shield/email-address-obfuscation/).

The website also uses [Cloudflare Web Analytics](https://www.cloudflare.com/en-gb/web-analytics/).
